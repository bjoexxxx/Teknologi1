Forklar hvad vi kan bruge følgende talsystemer til samt deres navne:

16 tal systemet
10 tal systemet
2 tal systemet
Svar:
16 systemet eller hexadecimal er et system der går fra 0-G heraf 16 tal
det bruges primært til at opbevarer store mængder af data på kortere form end binær.
10 tals systemet eller decimal der det system vi mennesker normalt bruger 0-9
dette bruges udelukkende for mennekskelig forståelse
2 tal systemet eller binær består af 0 og 1
dette sprog er det eneste maskiner forstår og alt data bliver derfor oversat til dette

Forklar hvordan man regner mellem binær og decimal:

binær har kun to tal og hvert tal i rækken venstre mod højre bliver derfor dobbelt så stort,
som forrige så 1 = 1, 2 = 10, 3 = 11, 4 = 100, 5 = 101 osv.

Forklar hvordan man regner mellem binær og hexadecimal:

samme måde som binær til decimal bare
1 = 1, 9 = 1001, a = 1010, b = 1011, c = 1100, d = 1101

Hvad bruges DNS til ift. Computer netværk?

DNS er Domain Name Server, og bruges til at findes din IP på nettet
f.eks hvis du skriver Facebook.com, går din maskine ikke ind og finder
addressen Facebook, men IP det er blive sat til at være synonym med
Facebook.com
